## Competitive landscape

| Category | Projects | Core approach | Membria focus |
| --- | --- | --- | --- |
| Centralized LLMs | OpenAI, Anthropic, Google | Cloud-hosted models with embedded knowledge | Local execution + decentralized memory + verifiable provenance |
| RAG / knowledge systems | Pinecone, Weaviate, LangChain | Vector search + static corpora | Reusable knowledge graph with incentives + permanent storage |
| Decentralized AI | Bittensor, Fetch.ai | Decentralized compute/agents | Knowledge-first infrastructure for lightweight models |
| Local AI tooling | Ollama, LM Studio | Run models locally | Augments local models with shared memory |
| Knowledge platforms | Wikipedia | Human-curated knowledge | Machine-native, verifiable, AI-optimized artifacts |

## Differentiation

**Technological:** Model-knowledge separation, local-first inference, verifiable artifacts (Arweave + Peaq), reusable intelligence.

**Commercial:** Pay-per-distillation, cache-first cost structure, user-controlled data, open-source client, decentralized backend.

## Partnerships & strategy

### Key partners

* **R&D:** FSID, University of Amsterdam, University of Luxembourg for AI memory research.
* **Enterprise:** Mistral, Snowflake, FalkorDB, Databricks, NVIDIA (Inception program) for infrastructure + edge AI deployments.
* **Blockchain:** Arweave (storage), Peaq Protocol (coordination), Fetch.ai (knowledge enhancement and ASI Alliance).
