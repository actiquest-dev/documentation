## Roles

| Role | Primary use | Key screens |
| --- | --- | --- |
| Partner | Review signals, approve decisions, governance | Decision Surface, DBB |
| Manager | Create decision briefs, manage team work | DBB, Search |
| Associate | Find precedents, document decisions | Search, DBB |
| Admin | Configure agents, manage users | Settings |

## Core workflows

**1) Morning check (partner, 5 min)**
- Open Decision Surface
- Review signals needing attention
- Approve pending decisions

**2) New matter setup (manager, 15 min)**
- Create a new matter
- Review similar precedents
- Set decision framework and assign team

**3) Decision documentation (associate, 10 min)**
- Open Decision Brief Builder
- Enter decision and context
- Add reasoning with AI prompts
- Submit for partner approval

**4) Precedent search (anyone, 2 min)**
- Ask Butler for similar cases
- Review decisions and reasoning
- Apply to current situation
