# Implementation

## Onboarding (6 weeks)

| Week | Activity |
| --- | --- |
| 1 | Kickoff, data inventory, integration setup |
| 2-3 | Historical decision import, precedent seeding |
| 4 | User training, workflow configuration |
| 5 | Pilot with one practice area |
| 6 | Full rollout, success criteria review |

## Success criteria (90 days)

- 100 or more decisions documented
- 80 percent of users active monthly
- 5 or more precedents reused
- At least 1 drift detection acted upon
- Partner NPS above 40
