# Membria product documentation

Membria is a family of products that share a common runtime philosophy: local-first reasoning where possible, explicit memory, and selective escalation. This documentation is organized by product.

## How to use these docs

Each product branch includes:
- Overview of the product and audience
- Getting started
- Core concepts
- Deployment and hosting
- Security and privacy

If you are unsure where to start, begin with the CE overview and then follow the branch that matches your deployment needs.
